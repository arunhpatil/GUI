Comprehensive analysis of small RNA sequencing data.

Author(s): Arun H. Patil and Marc K. Halushka

Laboratory: The Halushka Laboratory, Johns Hopkins University

miRge3.0:
miRge3.0 is a small RNA RNA-seq analysis pipeline, developed in a python package wrapped around an Electron framework to provide users an interactive graphical interface. This python package is also available for command line interface, for users who enjoy working on a Linux platform.

miRge3.0 is an updated version developed in python v3.8 which was proceeded by miRge2.0 (python 2.7) and miRge (PERL and python 2.7).
